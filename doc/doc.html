Popis implementácie

Projekt je implementovaný podľa modelu MVC.

Súbory s príponami .blade.php predstavujúce časť View. V týchto súboroch sa nacházajú konktétne zobrazenia stránok. Jednotlivé stránky sú výstižne pomenované podľa toho, čo konkrétne majú zobrazovť.

Časť Model predstavuje prácu s databázov. Tu sa nachádzajú čisto operácie, ktoré je nutné vykonať v databáze. Ide o dotazy ako napríklad select, delete, update a ďalšie. Súbory je možné nájsť v zložek App\Models. Každý model je pomenový podľa tabuľky, nad ktorou vykonáva operácie. Jednotlivé metódy modulov sú volané pomocou controlleru.

Súbory v zložke App\Http\Controllers predstavujú Controller časť MVC modelu. Tu sa vykonáva všetka komunikácia a spracovanie dát medzi View a Model časťou. Controllery sú opäť výstižne pomenované. Napríklad AnimalController.php slúži pre správu zvierat, zatiaľ čo ExaminaionController slúži k spracovaniu dát vyšetrení a iné.

Vzhľad a vytvorenie databáze zabezpečujú takzvané migrácie. Pomocou príkazu php artisan migrate, sa zo zložky database\migrations načítajú schémy databázy, pomocou ktorých je databáza vytvorená.

Prístup medzi jednotlivými používateľmi a zabránenie prístupu do sekcie, na ktorú iný používateľ nemá prístup je riešená pomocou rozšírenia middleware. Ide o rozšírenie, ktoré podľa autentifikovaného používeľa sprístupňuje len určitú časť prístupných ciest. Kontrola autentifuikácie používeľa pri prístupe k určitej ceste prebieha v častiach App\Http\Middleware. Povolené prístupové cesty sú definované pre každú rolu používateľa v súbore routes\web.php. Tento súbor slúži zároveň aj pre rozpoznanie dotazovaného pohľadu podľa linku a vyvolanie konkrétneho kontroleru.

Inštalácia pre localhost

  • Git clone / skopírovanie súborov do zložky projektu
  • Inštalácia a spustenie DB serveru, napríklad XAMPP s MariaDB min. verzia 10.4.25
  • Vytvoríť súbor .env pre prepojenie s DB systémom. Správne nastavenie parametrov. Ukažkový súbor example.env
  • Prípadné vloženie dummy dát vo forme .sql (priložný script) cez rozhranie phpmyadmin
  • Pomocou PowerShell (Windows) alebo terminálu (Linux) inštalácia php 8.1, composer, node.js min. 16.xx.x
  • Inicializácia databázi pomocou príkazu php artisan:migrate
  • V jednom terminály spusiť npm príkazom: npm install & npm run dev
  • V druhom terminály spustiť artisan príkazom: php artisan serve
  • Prístup ku stránke pomocou localhost, prípadne: http://127.0.0.1:8000/
Projekt nie je možné spustiť iba z odovzdaných súborov, nakoľko neobsahujú všetky knižnice, dependencies a iné controllery vygenerované a požadované priamo Laravellom. Rovnako neobsahujú bootstrap knižnice potrebné pre korektné zobrazenie stránky. Celý projekt je možné po požiadaní zobraziť na GitHub.
